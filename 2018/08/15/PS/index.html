PhotoShop手艺活

写在前面

自从入坑摄影后，PS这门手艺活也是不得不学了。但是真的不想去买课（留下了贫穷的泪水～）最后还是跟着B站啃下来了hhhh

记录一些常用的快捷键吧。

基础操作

新建文档：Command + N
打开现有文档：Command + O
保存文档：Command + S
另存为：Shift + Command + S
关闭文档：Command + W
撤销：Command + Z
重做：Shift + Command + Z
复制：Command + C
剪切：Command + X
粘贴：Command + V
全选：Command + A
取消选择：Command + D
放大：Command + +
缩小：Command + -
转到图层：Command + J
向下合并图层：Command + E
全屏预览：F
打开调整图层对话框：Command + L
打开色彩平衡对话框：Command + B
打开曲线对话框：Command + M

常用操作

启用自由变换工具：Command + T
启用自由变换工具并保持纵横比：Option + Command + T
启用变形工具：Command + Shift + T
启用图层蒙版：Option + 点击图层蒙版缩略图
显示/隐藏图层蒙版：Shift + 点击图层蒙版缩略图
快速选择工具：W
启用笔刷工具：B
快速启用橡皮擦工具：E
启用裁剪工具：C
启用移动工具：V
启用矩形选框工具：M
启用椭圆选框工具：M，然后按住Shift键绘制
启用套索工具：L
启用快速选择工具：W
启用魔法棒工具：W，然后按住Shift键单击选择更多颜色
启用放大镜工具：Z
启用手形工具：H

进阶操作

启用图层样式对话框：Option + 点击图层样式按钮
显示/隐藏所有面板：Tab
快速启用/禁用参考线：Command + ;
快速启用/禁用网格：Command + ‘
启用历史记录画笔工具：Y
启用污点修复画笔工具：J
启用渐变工具：G
启用填充工具：G，然后按住Option键填充选定区域
启用文本工具：T
启用路径选择工具：A
启用直线选择工具：A，然后按住Option键
启用自定义形状工具：U
启用图像修复工具：S
启用切片工具：K
